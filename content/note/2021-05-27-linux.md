## Networking
### lsof
```bash
$ lsof -nP -i4TCP:1080 -sTCP:LISTEN
$ lsof -nP -i4TCP:1080 -sTCP:ESTABLISHED
$ lsof -iTCP -sTCP:LISTEN -n -P
# -a means AND in lsof
$ lsof -itcp -a -p "1280595"
```

### tcpdump
```bash
$ sudo tcpdump -i all host 127.0.0.1 and port 1080 -w cap1.pcap
```

### iperf
```bash
# server
iperf3 -s

# client
iperf3 -c 192.168.2.1
```
